# Premium Portfolio Website - Next.js

Production-ready personal portfolio for Developer / Freelancer / AI Automation Developer.

## Tech Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- next-themes
- lucide-react

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build for Production

```bash
npm run build
npm run start
```

## Deploy on Vercel

1. Push this project to GitHub.
2. Import the repository in Vercel.
3. Framework Preset: Next.js.
4. Build Command: `npm run build`.
5. Output Directory: leave default.

## Customize

Edit `lib/data.ts` to change your name, links, skills, projects, services, testimonials, and experience.

Place your resume file in `public/resume.pdf` to enable the Download Resume button.
